About Study Buddy

Study Buddy originally started in 2017 by me as a project which was recommended by someone who I will not expose the person's identity.

Who is Study Buddy for?

Study Buddy is primarily for students as there are some extremely useful things in the app such as today's word of the day, todo lists and last but not least, an upcoming Google Classroom integration (assuming I can find documentation).

What's the current status of Study Buddy?

Currently, Study Buddy is still a work in progress and new features will be (eventually) released to the public. However, it's been at least probably less than half a year since the website has been updated to support the new database.

I'm a developer. Can I build the source code? Where is it?

Hi developer! Firstly, the source code is available at Chan4077/StudyBuddy, while the source code for the web version is at Chan4077/StudyBuddy-web

I would like to contribute to this project! How do I do so?

Thanks. I appreciate changes made by others. I accept PRs (pull requests) for those who actually use Github for open-source projects.
